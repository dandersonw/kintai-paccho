# kintai-paccho
Slackから勤怠入力するSlack Botだぱっちょ。

[KING OF TIME](https://www.kingtime.jp/) のみに対応しています。


## 使い方

```
$ git clone git@github.com:showwin/kintai-paccho.git
$ cd kintai-paccho
$ nohup python run.py &
```
